Plot Utils is a Minecraft Bedrock addon designed to make plot control and assignment easier for realms and servers. It is not focused on advanced systems such as plot purchasing, automatic claiming, or automatic generation. Instead, it is built to help administrators **create, assign, manage, and protect plots** in a cleaner and faster way.

The addon works completely in-game using the official Script API, with a visual hotbar toolbox, simple configuration menus, and a clean workflow made to feel polished, versatile, and easy to use.

---

## Features

- **Protected plots** — Create plot areas with a name, owner, dimension data, and configurable permissions.
- **Plot groups** — Group multiple plots inside a larger protected region. Groups can also protect outside areas such as roads, paths, and shared zones.
- **Hotbar toolbox** — A versatile menu with guided tools that lets administrators create, manage, and configure plots without any commands.
- **Configurable** — Several parameters allow administrators to adjust permissions, visual settings, and addon behavior.
- **Multilingual** — Supports English and Spanish (es_MX).

## How to create a plot

This is a simple showcase of the basic plot creation workflow.

### 1. Get the Plot Utils toolbox

Open your inventory, search in the items tab, and take the toolbox called **Plot Utils**.

<div align="center">
<img src="docs/images/inventory.gif" width="820" alt="Get the toolbox">
</div>

### 2. Open the toolbox

Place the toolbox in your hotbar and interact with it.

<div align="center">
<img src="docs/images/toolbox.gif" width="820" alt="Open the toolbox">
</div>

### 3. Select Create Plot

Interact with the **Create Plot** item from the toolbox.

<div align="center">
<img src="docs/images/create.gif" width="820" alt="Select Create Plot">
</div>

### 4. Select the plot limits

Use the **Selection Wand** to mark the boundaries of your plot.

<div align="center">
<img src="docs/images/plot_bounds.gif" width="820" alt="Select plot limits">
</div>

### 5. Finish the plot

Use the finish item, enter the plot name, and assign the owner.

<div align="center">
<img src="docs/images/finish.gif" width="820" alt="Finish the plot">
</div>

> [!IMPORTANT]
> This is only a small showcase of the basic workflow. Plot Utils includes more configuration options, permissions, visual tools, and management features that can be adjusted depending on how your realm or server is organized.
